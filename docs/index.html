Silakka54 - RP2040 based split keyboard

Silakka54 is a RP2040 Zero based 54-key column staggered split keyboard. PCB uses hotswap sockets. Design is inspired from REVIUNG41 and Corne keyboards.

PCB is designed for MX style key switches. 5-pin switches are supported. You can find files for the PCB in the repository. Firmware is based on QMK and there is Vial version, for Vial GUI software support. You can find the latest precompiled versions on the releases section.

Notice about pre-built keyboards

There are now pre-built silakka54 keyboards on the market. I am not affiliated with those in any way. If you have bought one, I recommend updating to the latest firmware.
